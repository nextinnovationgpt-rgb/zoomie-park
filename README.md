# Zoomie Park - 24時間営業屋内ドッグラン＆ラウンジ

## プロジェクト概要

**Zoomie Park** は、「愛犬の"走り回りたい"本能を、24時間いつでも満たせる場所」として、奈良県北葛城郡上牧町に誕生した地域初の「24時間営業・日中はスタッフ在中、夜間は無人運営」の屋内ドッグラン＆ラウンジ施設の紹介ウェブサイトです。

年々厳しさを増す夏の暑さや、変わりゆく空模様。そして、共働きや在宅ワークといったライフスタイルの変化。

「思い切り走らせてあげたいけれど、場所がない」「仕事が終わる頃には、もう外は暗い」

そんな、愛犬を想うがゆえの葛藤を抱える飼い主様が増えています。

だからこそ、ここは単なるドッグランではありません。天候や時間を忘れ、愛犬は本能のままに駆け回り、飼い主様はラウンジで自分だけの時間を慈しむ。

そんな「愛犬と人の、新しい幸せのカタチ」を、ここ奈良・上牧町からお届けします。

## 🎯 主な特徴

### 1. 24時間営業・日中スタッフ在中・夜間無人運営
- 9:00-20:00はスタッフが常駐し、安心サポート
- 20:00-翌8:00はIT技術によるスマートな無人運営
- スマホで完結する予約・決済システム
- 24時間稼働の監視カメラによるセキュリティ

### 2. 天候に左右されない快適空間
- 空調完備の清潔な室内空間
- 滑りにくくクッション性の高い床材
- 小型犬やシニア犬にも優しい設計

### 3. 飼い主様のためのラウンジ機能
- Wi-Fi、電源、マッサージチェア完備
- コーヒーを飲みながらのリラックスタイム
- 在宅ワーク中の利用も可能

### 5. 会員様限定のプレミアムサービス
- 会員様限定フリードリンクサービス
- コーラ、コーヒー、ジュース、天然水など
- 優雅なひとときを提供する特別な空間

### 6. サブスクリプション型料金体系
- **平日限定会員** - 月額4,780円（平日日中メイン利用、主婦層・リモートワーカー向け）
- **全日会員** - 月額6,480円（24時間・年中無休利用、ファミリー層向け）
- **ビジター利用** - 1,500円/頭（お試し利用・遠方からの来店向け）
- すべての会員プランにフリードリンク付き
